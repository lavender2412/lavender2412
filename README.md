## 🚀 Featured Projects

### 🏥 [ClinsightAI](https://github.com/lavender2412/Code_Blooded_ClinsightAI)
Multi-task NLP pipeline analyzing hospital patient reviews using LDA topic modeling and Ridge Regression on TF-IDF features, improving R² from 0.046 to 0.302. Produces per-review theme contribution scores, systemic issue classification, and a priority-ranked action roadmap with KPI targets — surfaced through a Streamlit dashboard.

### 📈 [User Journey Analysis](https://github.com/lavender2412/User_Journey_Analysis)
End-to-end ML pipeline using Python and PySpark for multi-session user conversion prediction. Trained Logistic Regression, Random Forest, and XGBoost models achieving ROC-AUC of 0.84. Deployed as a Flask REST API on AWS Elastic Beanstalk with GitHub Actions CI/CD.

### 🎥 [Suspicious Activity Detection](https://github.com/lavender2412/Suspicious_Activity_Detection)
Dual-module real-time surveillance system combining a spatiotemporal autoencoder (Conv3D + ConvLSTM2D + Conv3DTranspose) for body gesture anomaly detection and a MediaPipe face mesh module with PnP head pose estimation, with automated Twilio SMS alerts. **Published and presented at IEEE TENCON 2023** — [View Paper](https://ieeexplore.ieee.org/document/10322454)
